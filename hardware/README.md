# Hardware

Hardware notes, pinouts, schematics, and wiring references belong here.

- `shield-v2-pinout.md`: Teensy, S32K SPI, and ESP32 shield pin mapping.
- `reference-documents/nxp-car-shield-v2-schematic.pdf`: exported four-page
  PCB schematic for the NXP car shield.

Keep exported PDFs, datasheets, and other non-code hardware references inside
`reference-documents/`. Keep handwritten wiring summaries and decisions as
Markdown files directly in `hardware/`.
